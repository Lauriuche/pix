# 💳 POS PIX PWA

Sistema simples de ponto de venda (PDV) com geração de pagamento via PIX usando Mercado Pago.

## 🚀 Funcionalidades

- Digitar valor
- Gerar QR Code PIX
- Verificação automática de pagamento
- Status em tempo real
- Interface estilo maquininha
- PWA (instalável)
